# Production Process

## Process Steps

1. **File import/setup** — open the Elliott-supplied production-ready DWG, send to cutter. ~5 min mechanical. NOT billable labor (Core Rule 7 / `PRICING_RULES.md` §22) — Elliott provides production-ready DWG files for every item on this account without exception.
2. **Cut** — Roland SG3-300 on 24" Cardinal Red stock. 3 labels across (3 × 8" = 24" across the 24" roll, zero clearance per the Round 0 layout assumption). Single continuous cut path of the "G50" character set, repeated 3-up per row.
3. **Weed** — remove excess vinyl around the 3 alphanumeric characters (G, 5, 0). Low complexity: G has one partial open counter, 5 has no enclosed counter, 0 has one clean circular counter. Simpler geometry than the account's E190/E160 model designations (5 characters each).
4. **Mask** — apply TransferRite 582U transfer tape over weeded characters. 24" tape matches 24" vinyl exactly.
5. **Inspect** — verify cut accuracy, no lifted edges, character legibility across the 21" width.
6. **Package** — flat, masked with transfer tape carrier, ready for hand-application on crane body/boom.

## Production Notes

- **Lamination passes:** 0 (cut vinyl, no print)
- **Cut runs:** 1
- **Equipment:** Roland SG3-300 (24" Cardinal Red stock; 28" max cut width — well within tolerance)
- **Weeding complexity:** Low. 3 simple block alphanumeric characters, one partial counter (G) and one clean circular counter (0) — the simplest character set of any cut vinyl model-designation item on the account to date.
- **Cut feed length:** 21" per row of 3 labels (as filed, no spacing allocation) — see Nesting and Material Cost for the account-standard corrected feed length (27").
