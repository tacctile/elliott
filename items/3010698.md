# Nesting and Material Cost

## 90-Degree Rotation Geometry

The label is **33.4375" wide × 6.8125" tall** when installed. During production the label is **rotated 90 degrees on the roll** so that:

- **6.8125" (label height) feeds ACROSS the roll width** — enables multi-across nesting
- **33.4375" (label width) feeds ALONG the roll length** — becomes the cut length per row

This is the same length-based, length-feeds-along-roll convention used by every other length-based cut vinyl item on this account (1205720, 3010701, 3010704, 1146650).

## Corrected Method — Replaces Round 0 Calculator Brief

The Round 0 calculator brief filed this item's tape cost using the deprecated area-based formula: 1.582 sq ft × $0.5911/sq ft = $0.9351/label. `materials/transferrite-582u.md`'s 2026-06-09 method note (audit D3) already flagged this exact formula as a pseudo-rate ($/yd ÷ roll-width-ft, not a true area rate) that overstates tape cost ~2.7× — the same correction already applied to the small-format cluster (1205720, 1186310, 3017435, 3018378). The cluster and this size class are harmonized to the **length-based method**: `feed length (label length + 6" spacing) in yd × $1.1821/yd ÷ labels per row`. Per session instruction, this correction applies to this item's filing only — the calculator's tape-cost formula itself is not edited this session (see Notes and Warnings, process finding).

# Production Process

## Process Steps

1. **File import/setup** — open the Elliott-supplied production-ready file, send to cutter. ~5 min mechanical. NOT billable labor (see `PRICING_RULES.md` §22).
2. **Cut** — Roland SG3-300 on 24" Cardinal Red stock. 3 labels across (3 × 6.8125" = 20.4375" across 24" roll, 3.5625" clearance). Single continuous cut path of the "ELLIOTT" wordmark, repeated 3-up per row.
3. **Weed** — remove excess vinyl around the 7-character "ELLIOTT" wordmark. Confirmed via direct artwork review: six open block-letter forms (E, L, L, I, T, T) with straight cuts and no interior weeding, plus one clean circular O counter — no serifs, no thin stems, no nested or stacked cutouts. The 33.4375" weed span across the row of 3 labels is the dominant time factor.
4. **Mask** — apply TransferRite 582U transfer tape over weeded characters. 24" tape matches 24" vinyl exactly.
5. **Inspect** — verify cut accuracy, no lifted edges, character legibility across the 33.4375" width.
6. **Package** — flat, masked with transfer tape carrier, ready for hand-application on crane body/boom.
