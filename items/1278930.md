# Production Process

2. Print — Roland SG3-300, full-color eco-solvent on Orajet 3951. All 3 in one run.
3. Laminate — 1-mil polyester. **ONE pass — all 3 fit.**
4. Contour cut — **ONE run** following registration marks. R.25 corners.
5. Weed — remove excess around all 3 labels
6. Inspect — print quality, lam adhesion, cut accuracy, label-to-label consistency
7. Package — collate A, B, C as matched set
