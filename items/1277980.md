# Production Process

Identical to program peer P/N 1277970.

1. **File prep** — import 1277980 print/cut file, set up registration marks, define kiss-cut path (~15-20 min for this unique design).
2. **Print** — Roland SG3-300, full-color eco-solvent on Orajet 3951. Nested with the other 3 program designs in a single small print area.
3. **Laminate** — 1-mil clear polyester, single pass through the 13.5" laminator.
4. **Kiss cut** — Roland contour cut, face stock only, liner intact. Slit liner across center per drawing spec.
5. **Inspect** — print quality, laminate adhesion, kiss-cut depth, text legibility.
6. **Package** — flat, on liner, ready for hand application.
