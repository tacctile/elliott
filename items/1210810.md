# Nesting and Material Cost

### Laminator Pass Count

Both the 10.5" width and 4.0" height dimensions are well within the 13.5" polyester laminator. Single lamination pass in either orientation.

- **Preferred orientation (4" across, 10.5" feed direction):** 4" across the 13.5" laminator → 3 labels fit side by side (3 × 4" = 12.0" ≤ 13.5"). Efficient for batch runs.
- **Alternate orientation (10.5" across, 4" feed direction):** 10.5" across the 13.5" laminator → 1 label per pass width (3.0" clearance). Less efficient for volume runs.

**Single lamination pass confirmed.**

### Print Layout and Nesting

Label: 10.5" W × 4" H. Roland SG3-300 max print width: 28".

- **Orientation A (10.5" across Roland, 4" tall):** 10.5" across 28" → 2 labels across (2 × 10.5" = 21" ≤ 28"). 4" feed direction per row.
- **Orientation B (4" across Roland, 10.5" tall):** 4" across 28" → 7 labels across (7 × 4" = 28" = max). 10.5" feed direction per row.

At production volume (30 labels/run): Orientation B is optimal — 7 labels across, 5 rows = 35 labels per print run. Very efficient nesting for this label size.
